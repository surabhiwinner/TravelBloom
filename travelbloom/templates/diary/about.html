🧭 About SmartVoyager

SmartVoyager is your intelligent travel companion — a dynamic trip planner designed to help you discover, plan, and manage trips with ease. Whether you're solo backpacking or planning a family getaway, SmartVoyager makes your journey smarter.

🌟 Key Features

  • 📍 Smart Trip Planner: Search and filter places by city and category (attractions, hotels, transit).
  • 🗺️ Optimized Route Builder: Automatically sort your stops for best travel efficiency.
  • 📅 Saved Trips Dashboard: Plan, edit, or resume trips — grouped as Planned, Ongoing, and Completed.
  • 📊 Interactive Maps: View all stops on Google Maps with live progress tracking.
  • 📌 Visited Places Detection: Track places you’ve visited in real time using GPS proximity.
  • ✍️ Travel Diary & Blogs: Share travel stories, post images, and connect with fellow explorers.
  • 🤖 AI Travel Assistant (Coming Soon): Chat with Gemini AI to get personalized suggestions and instant answers.

🛠️ Tech Stack

SmartVoyager is built using modern technologies including:

  • ⚙️ Django Framework (Python)
  • 🗺️ Google Maps JavaScript & Places API
  • 🧩 HTML, CSS, JavaScript (Bootstrap 5)
  • 🧠 Gemini AI (for intelligent chatbot)
  • 🛢️ SQLite / PostgreSQL
  • 📍 Real-time browser-based geolocation
